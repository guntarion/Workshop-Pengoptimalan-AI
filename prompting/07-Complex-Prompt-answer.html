Konsep Complex Prompting

Complex Prompting adalah teknik menyusun prompt yang terstruktur dan komprehensif untuk mendapatkan hasil yang lebih akurat dan konsisten dari AI. Teknik ini menggabungkan beberapa elemen prompt engineering seperti:

  • Role dan Context Setting
  • Format Output yang Terstruktur
  • Step-by-Step Instructions
  • Constraints dan Rules
  • Examples dan Few-Shot Learning

Tips Complex Prompting

  • Mulai dengan mendefinisikan peran dan konteks yang jelas
  • Berikan format output yang terstruktur (JSON, XML, dll)
  • Sertakan contoh yang relevan dengan kasus
  • Tetapkan batasan dan aturan yang spesifik
  • Gunakan tag untuk mengorganisir bagian-bagian prompt
  • Verifikasi output dengan kriteria yang telah ditetapkan

Latihan Complex Prompting

HRD

"Analisis hasil assessment kandidat dan berikan rekomendasi keputusan rekrutmen dengan pertimbangan kompetensi teknis dan soft skills." 
<system>
Anda adalah HR Specialist yang berpengalaman dalam melakukan assessment kandidat untuk posisi teknis di pembangkit listrik.
</system>

<context>
Analisis harus mencakup evaluasi kompetensi teknis dan soft skills berdasarkan kriteria yang ditentukan, dengan output berupa rekomendasi terstruktur.
</context>

<format>
{
  "kandidat_info": {
    "nama": string,
    "posisi": string,
    "pengalaman": string
  },
  "assessment_scores": {
    "technical": {
      "score": number,

      "gaps": string[]
    },
    "soft_skills": {
      "score": number,
      "strengths": string[],
      "gaps": string[]
    }
  },
  "rekomendasi": {
    "keputusan": "HIRE|CONSIDER|REJECT",
    "justifikasi": string[],
    "development_areas": string[]
  }
}
</format>

Supply Chain

Akuntansi

"Analisis variance biaya operasional unit pembangkit dan berikan rekomendasi pengendalian biaya." 
<system>
Anda adalah Cost Controller yang berpengalaman dalam analisis biaya operasional pembangkit listrik.
</system>

<context>
Analisis harus mengidentifikasi root cause dari variansi biaya, dampak terhadap KPI, dan rekomendasi pengendalian yang actionable.
</context>

<format>
{
  "period_info": {
    "unit": string,
    "period": string,
    "baseline": string
  },
  "cost_variance": {
    "total_variance": {
      "amount": number,
      "percentage": number,
      "trend": "FAVORABLE|UNFAVORABLE"
    },
    "breakdown": [
      {
        "category": string,
        "variance": number,
        "contribution": number,
        "root_causes": string[]
      }
    ]
  },
  "kpi_impact": {
    "cost_per_kwh": {
      "target": number,
      "actual": number,
      "variance": number
    },
    "other_metrics": [
      {
        "name": string,
        "impact": string,
        "severity": "HIGH|MEDIUM|LOW"
      }
    ]
  },
  "recommendations": {
    "immediate_actions": string[],
    "long_term_measures": string[],
    "monitoring_points": string[]
  }
}
</format>
