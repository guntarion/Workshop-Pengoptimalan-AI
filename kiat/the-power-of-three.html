Contoh Implementasi: Sistem Monitoring Pembangkit Listrik

Contoh Prompt yang Baik:

"Sebagai system architect di pembangkit listrik tenaga uap 500MW, saya membutuhkan solusi monitoring realtime untuk efisiensi boiler, konsumsi bahan bakar, temperature turbin, dan vibrasi bearing. Berikan 3 variasi arsitektur dengan pertimbangan reliability, cost, maintenance, dan scalability."

Arsitektur Tradisional

  • SCADA based system
  • Database terpusat
  • HMI berbasis desktop
  • Komunikasi industrial ethernet

Arsitektur Modern

  • IoT sensors
  • Cloud based storage
  • Web based dashboard
  • Real-time analytics

Arsitektur Hybrid

  • Edge computing di plant
  • Redundant storage
  • Mobile + desktop interface
  • AI predictive maintenance

Implementasi Praktis

Persiapan Prompt yang Terstruktur

Contoh Struktur Prompt:

Role: Senior Engineer Pembangkit
Context: Optimasi sistem predictive maintenance turbin gas
Constraint: Budget 1M USD, implementasi max 6 bulan
Request: Berikan 3 variasi solusi dengan mempertimbangkan:
- Accuracy prediksi
- Speed implementasi
- Maintenance cost
- Team capability
  • Definisikan konteks dengan jelas
  • Tentukan kriteria/constraint spesifik
  • Minta eksplisit 3 variasi dalam prompt
  • Berikan parameter pembanding

Evaluasi & Iterasi Respon

ITERASI 1: Initial Request

"Berikan 3 variasi sistem early warning untuk vibrasi berlebih pada turbin"

ITERASI 2: Refined Request

"Dari 3 variasi sebelumnya, kita focus ke IoT based solution. Berikan 3 variasi detail arsitektur IoT dengan trade-off berbeda antara cost vs reliability"

ITERASI 3: Final Specification

"Untuk arsitektur IoT terpilih, berikan 3 variasi detail implementasi dengan:
- Tech stack berbeda
- Timeline implementasi
- Resource requirement
- Cost breakdown"
