Soal 1: Analisis Statistik Deskriptif & Efisiensi Pembangkit

Sebuah perusahaan listrik memiliki 3 pembangkit dengan data operasional harian selama periode 2020-2024. Data tersedia dalam file Analisis-Data-Operasional_power_plant_data.csv.

Tugas:

  1. Lakukan analisis statistik deskriptif untuk parameter berikut:
    • Beban pembangkit (load_mw)
    • Efisiensi (efficiency)
    • Emisi (emissions_tons)
  2. Uji hipotesis: Apakah ada perbedaan signifikan antara efisiensi pembangkit gas (Plant_2) dengan target efisiensi 42% (α = 0.05)?
  3. Visualisasikan distribusi efisiensi untuk ketiga pembangkit dalam bentuk box plot.

Petunjuk: Gunakan pendekatan statistik yang sesuai untuk menguji hipotesis (t-test atau z-test). Pertimbangkan penggunaan library seperti scipy.stats untuk analisis statistik.

Soal 2: Analisis Korelasi & Prediksi Kinerja

Menggunakan dataset yang sama, analisis hubungan antara berbagai parameter operasional pembangkit.

Tugas:

  1. Analisis korelasi antara:
    • Beban pembangkit dengan tingkat emisi
    • Temperatur ambient dengan efisiensi
    • Beban pembangkit dengan biaya operasi
  2. Buat model regresi untuk memprediksi:
    • Emisi berdasarkan beban pembangkit
    • Biaya operasi berdasarkan beban dan efisiensi
  3. Evaluasi akurasi model menggunakan R² dan RMSE

Petunjuk: Pertimbangkan penggunaan regresi linear berganda untuk prediksi biaya operasi. Lakukan normalisasi data jika diperlukan.

Soal 3: Analisis Time Series & Optimasi Operasional

Analisis pola temporal dari operasi pembangkit dan buat rekomendasi optimasi.

Tugas:

  1. Analisis time series:
    • Identifikasi pola musiman beban pembangkit
    • Analisis tren efisiensi jangka panjang
    • Deteksi anomali dalam data operasional
  2. Buat peramalan untuk 3 bulan ke depan:
    • Beban pembangkit
    • Biaya operasional
    • Revenue
  3. Berikan rekomendasi untuk optimasi operasional berdasarkan hasil analisis

Petunjuk: Gunakan teknik dekomposisi time series untuk memisahkan komponen trend, seasonal, dan residual. Pertimbangkan penggunaan model SARIMA atau Prophet untuk peramalan.

Bonus: Analisis Profitabilitas

Analisis profitabilitas pembangkit dan faktor-faktor yang mempengaruhinya.

Tugas:

  1. Hitung dan bandingkan profitabilitas ketiga pembangkit
  2. Identifikasi faktor-faktor yang paling mempengaruhi profitabilitas
  3. Buat dashboard visual untuk monitoring kinerja finansial

Petunjuk: Gunakan visualisasi interaktif seperti Plotly atau Bokeh untuk membuat dashboard yang informatif.
