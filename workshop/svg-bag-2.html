1

Pressure dan Temperature Gauges

Membuat gauge meter untuk monitoring tekanan dan temperatur secara real-time.

Tolong buatkan SVG gauge meter untuk monitoring tekanan boiler dengan spesifikasi berikut: Elemen gauge: - Semicircular dial (180 derajat) - Range: 0-100 bar - Major ticks setiap 10 unit - Minor ticks setiap 2 unit - Needle dengan pivot point - Current value display Dimensi dan Layout: - ViewBox: 300x200 - Radius: 80px - Center point: 150,150 - Scale arc: stroke-width 4px Visual elements: - Color zones: * 0-60: hijau (#34D399) * 61-80: kuning (#FBBF24) * 81-100: merah (#EF4444) - Gradient backgrounds - Digital readout - Unit label (bar) Interaktivitas: - Smooth needle animation - Hover tooltips - Value display update - Warning indicator

Struktur Prompt:

  1. Definisi elemen gauge
  2. Spesifikasi dimensi dan posisi
  3. Pembagian zona nilai
  4. Interaktivitas dan animasi
Gunakan CSS transitions untuk animasi needle yang smooth dan natural.
2

Status Indicators dan Controls

Membuat indikator status dan kontrol panel untuk monitoring sistem.

Tolong buatkan set SVG status indicators untuk control panel dengan spesifikasi berikut: Indikator yang dibutuhkan: - Running/Stopped status - Warning level indicator - System health meter - Connection status - Load level display Layout per indicator: - ViewBox: 100x100 - Circular design - Status text below - Icon di tengah Style elements: - Active state colors: * Running: #10B981 * Warning: #F59E0B * Critical: #DC2626 * Standby: #6B7280 - Pulsing effect untuk status aktif - Gradient overlays - Inner/outer rings Animations: - Smooth state transitions - Pulse effect untuk alerts - Rotate untuk loading - Fade untuk status change

Struktur Prompt:

  1. Definisi tipe indikator
  2. Layout dan dimensi
  3. State dan color coding
  4. Animasi dan transisi

Sample State Data:

indicator,state,value Running,active,100 Warning,inactive,0 Health,warning,75 Connection,active,100 Load,critical,90
Implementasikan debouncing untuk status updates untuk menghindari flicker pada rapid state changes.
