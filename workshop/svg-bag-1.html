1

Safety Icons dan Warning Signs

Membuat icon dan tanda peringatan yang efektif untuk area pembangkit listrik.

Tolong buatkan SVG icon warning untuk area bertegangan tinggi dengan spesifikasi berikut: Elemen utama: - Simbol petir di dalam segitiga peringatan - Teks "DANGER" dan "HIGH VOLTAGE" - Border ganda untuk emphasis Dimensi dan Layout: - ViewBox: 200x200 - Segitiga: tinggi 120px, lebar 140px - Stroke width: 3px untuk outline utama - Padding: 20px dari edge Style: - Warna: kuning (#FFD700) untuk background - Stroke: hitam (#000000) untuk outline - Petir: putih (#FFFFFF) dengan outline hitam - Text: sans-serif bold Efek visual: - Drop shadow untuk depth - Rounded corners (2px) pada segitiga - Gradient overlay untuk dimensi - Stroke-dasharray untuk inner border

Struktur Prompt:

  1. Definisi elemen visual utama
  2. Spesifikasi ukuran dan posisi
  3. Detail styling dan warna
  4. Efek visual tambahan
Gunakan shadow-opacity yang rendah (0.2-0.3) untuk efek subtle pada warning signs.
2

Technical Equipment Illustration

Membuat ilustrasi teknis untuk komponen pembangkit listrik.

Tolong buatkan ilustrasi teknis SVG untuk turbin generator dengan spesifikasi berikut: Komponen yang ditampilkan: - Rotor dengan blade detail - Stator housing - Bearing assemblies - Cooling system ports - Connection points Layout: - ViewBox: 400x300 - Cross-section view dari samping - Detail callouts untuk komponen kritis - Scale indicator Style elements: - Stroke width variasi (0.5-2px) - Dashed lines untuk hidden parts - Pattern fill untuk material types - Technical dimensioning Detailing: - Part numbers dan labels - Material indicators - Flow direction arrows - Assembly notes

Struktur Prompt:

  1. Identifikasi komponen utama
  2. Spesifikasi view dan layout
  3. Detail teknis dan dimensi
  4. Anotasi dan labeling
Gunakan layers yang terorganisir untuk memudahkan modifikasi komponen individual.
