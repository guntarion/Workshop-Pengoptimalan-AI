Performance Radar Chart

Membuat radar chart custom untuk visualisasi multiple performance metrics.

Tolong buatkan SVG radar chart untuk KPI monitoring dengan spesifikasi berikut: Metrics yang ditampilkan: - Efficiency (%) - Reliability (%) - Availability (%) - Response Time (ms) - Quality Score - Cost Effectiveness Layout specs: - ViewBox: 400x400 - Center point: 200,200 - Radius: 150px - 6 axes dengan labels - 5 level indicators per axis Visual elements: - Polygon area untuk current values - Reference polygon untuk targets - Grid lines untuk setiap level - Axis labels dengan units - Value indicators Styling: - Area fill: rgba(37, 99, 235, 0.2) - Stroke: #2563EB - Grid: rgba(0, 0, 0, 0.1) - Labels: #1E293B - Highlight points: #FFFFFF Interaktivity: - Hover tooltips per point - Click to highlight axis - Double click to isolate metric - Zoom capability

Struktur Prompt:

  1. Definisi metrics
  2. Layout specifications
  3. Visual styling
  4. Interactive features
