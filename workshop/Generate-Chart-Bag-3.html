Scatter Plot: Analisis Performa dan Efisiensi

Membuat scatter plot untuk menganalisis korelasi antara berbagai parameter operasional pembangkit.

Tolong buatkan scatter plot untuk analisis performa dengan spesifikasi berikut: Data yang ditampilkan: - Beban (MW) vs Efisiensi (%) - Heat Rate (kCal/kWh) - Konsumsi bahan bakar (ton/jam) - Data operasi per jam - Multiple units Elemen visual: - Points dengan variasi ukuran berdasarkan konsumsi - Sumbu X: Beban (MW) - Sumbu Y: Efisiensi (%) - Color coding untuk tiap unit - Trend lines - Reference zones untuk operasi optimal Styling: - Point opacity untuk density visualization - Clear axis labels dengan units - Grid untuk reference - Color scheme yang membedakan units - Trend line styling yang berbeda per unit Interaktivitas: - Zoom untuk detail area tertentu - Pan untuk explorasi data - Hover untuk detail point - Brush select untuk analisis cluster - Toggle trend lines - Filter berdasarkan unit atau periode

Struktur Prompt:

  1. Parameter correlation specs
  2. Multi-dimensional data display
  3. Trend analysis requirements
  4. Interactive exploration features

Sample Data (performance_data.csv):

Combined Chart: Dashboard Operasional Terintegrasi

Membuat dashboard yang mengkombinasikan berbagai jenis chart untuk monitoring komprehensif.

Tolong buatkan dashboard dengan kombinasi chart untuk monitoring operasional dengan spesifikasi berikut: Komponen yang ditampilkan: 1. Line Chart: - Tren beban real-time - Target vs Actual - Rolling average 2. Bar Chart: - Produksi harian per unit - Konsumsi bahan bakar - Side-by-side comparison 3. Area Chart: - Distribusi beban antar unit - Stack percentage view - Cumulative totals 4. Pie Chart: - Status unit (running/standby/maintenance) - Fuel mix distribution - Performance metrics Layout dan Integrasi: - Grid layout responsif - Shared color scheme - Synchronized tooltips - Common time range selector - Unified legend - Cross-chart highlighting Interaktivitas: - Global filters untuk periode - Drill-down capabilities - Export options - Custom view saving - Real-time updates - Alert indicators

Struktur Prompt:

  1. Multi-chart integration specs
  2. Layout dan organizational structure
  3. Cross-chart interaction design
  4. Data synchronization requirements
