Line Chart: Monitoring Efisiensi Pembangkit

Membuat grafik garis untuk memantau tren efisiensi pembangkit listrik dari waktu ke waktu.

Tolong buatkan visualisasi line chart untuk monitoring efisiensi pembangkit dengan spesifikasi berikut: Data yang ditampilkan: - Efisiensi thermal harian (%) - Heat rate (kCal/kWh) - Target efisiensi - Periode: 30 hari terakhir Elemen visual yang dibutuhkan: - Sumbu X: Tanggal - Sumbu Y kiri: Efisiensi (%) - Sumbu Y kanan: Heat rate - Legend untuk setiap line - Grid lines untuk memudahkan pembacaan - Tooltip yang menampilkan nilai detail Styling: - Warna berbeda untuk setiap line - Dashed line untuk target - Gradient area di bawah line efisiensi - Responsif untuk berbagai ukuran layar Interaktivitas: - Hover effect untuk menampilkan nilai detail - Click untuk highlight specific point - Zoom untuk range tertentu - Pan untuk melihat data historis

Struktur Prompt:

  1. Data requirements yang spesifik
  2. Elemen visual yang dibutuhkan
  3. Styling preferences
  4. Interaktivitas yang diinginkan

Sample Data (efficiency_data.csv):

date,efficiency,heat_rate,target_efficiency 2025-01-01,32.5,2650,33.0 2025-01-02,32.8,2645,33.0 2025-01-03,33.1,2640,33.0 2025-01-04,32.9,2642,33.0 2025-01-05,32.7,2648,33.0 [...lanjutkan untuk 30 hari]

Bar Chart: Analisis Konsumsi Bahan Bakar

Membuat grafik batang untuk menganalisis dan membandingkan konsumsi bahan bakar antar unit pembangkit.

Tolong buatkan visualisasi bar chart untuk analisis konsumsi bahan bakar dengan spesifikasi berikut: Data yang ditampilkan: - Konsumsi bahan bakar per unit (ton/hari) - Produksi listrik (MWh) - Specific fuel consumption (kg/kWh) - Perbandingan antar unit (Unit 1-4) - Data bulanan untuk tahun berjalan Elemen visual: - Grouped bars untuk konsumsi dan produksi - Overlay line untuk specific consumption - Sumbu X: Unit pembangkit - Sumbu Y kiri: Konsumsi & produksi - Sumbu Y kanan: Specific consumption - Legend untuk setiap metrik Styling: - Warna kontras untuk tiap grup bar - Pattern fill untuk membedakan metrik - Label nilai di atas setiap bar - Grid horizontal untuk pembacaan nilai Interaktivitas: - Tooltip multi-metrik - Click untuk filter unit - Sort berdasarkan berbagai metrik - Toggle visibility per metrik

Struktur Prompt:

  1. Spesifikasi data multi-metrik
  2. Layout visual yang jelas
  3. Styling untuk kemudahan pembandingan
  4. Interaktivitas untuk analisis

Sample Data (fuel_consumption.csv):

unit,date,fuel_consumption,power_production,specific_consumption Unit 1,2025-01,12500,45000,0.278 Unit 2,2025-01,13200,47500,0.278 Unit 3,2025-01,12800,46000,0.278 Unit 4,2025-01,12900,46500,0.277 [...lanjutkan untuk 12 bulan]
