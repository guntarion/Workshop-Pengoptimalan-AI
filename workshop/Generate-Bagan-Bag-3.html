Class Diagram: Sistem Monitoring Pembangkit

Membuat class diagram untuk sistem monitoring yang akan dikembangkan oleh vendor.

Tolong buatkan class diagram untuk sistem monitoring pembangkit menggunakan Mermaid dengan spesifikasi berikut: Classes: 1. Sensor Attributes: - sensorId: string - type: string - location: string - lastCalibration: date Methods: - readValue() - calibrate() - getStatus() 2. Equipment Attributes: - equipmentId: string - name: string - category: string - installationDate: date Methods: - startOperation() - stopOperation() - getOperatingHours() 3. MonitoringSystem Attributes: - systemId: string - status: string Methods: - collectData() - generateReport() - setAlarmThresholds() 4. Alarm Attributes: - alarmId: string - severity: enum - message: string Methods: - trigger() - acknowledge() - reset() 5. DataLogger Attributes: - loggerId: string - storageCapacity: number Methods: - logData() - exportData() - clearOldData() Relationships: - Equipment "1" *-- "many" Sensor - MonitoringSystem "1" *-- "many" Equipment - MonitoringSystem "1" *-- "many" Alarm - MonitoringSystem "1" -- "1" DataLogger Gunakan format: classDiagram dengan: - Proper visibility modifiers (+, -, #) - Relationship arrows - Method parameters - Interface definitions if needed

Struktur Prompt:

  1. Definisi classes dengan attributes dan methods
  2. Visibility modifiers untuk encapsulation
  3. Relasi antar classes
  4. Kardinalitas relationships
  5. Interface atau abstract classes bila diperlukan
Sesuaikan dengan kebutuhan spesifik sistem monitoring di fasilitas Anda, termasuk tipe sensor dan equipment yang digunakan.
2

Pie Chart: Analisis Distribusi Gangguan

Membuat pie chart untuk visualisasi distribusi jenis gangguan operasional.

Tolong buatkan pie chart untuk distribusi gangguan operasional menggunakan Mermaid dengan spesifikasi berikut: Judul: "Distribusi Gangguan Operasional Q1 2025" Data: - "Gangguan Mekanis" : 35 - "Gangguan Elektrikal" : 28 - "Masalah Instrumentasi" : 20 - "Human Error" : 12 - "Faktor Eksternal" : 5 Format tampilan: - Urutkan dari yang terbesar - Tampilkan persentase - Tambahkan warna yang kontras - Sertakan legenda Gunakan format: pie showData dengan: - title untuk judul - datavalue untuk nilai - Pemisahan segmen yang jelas

Struktur Prompt:

  1. Judul yang informatif
  2. Data dengan label dan nilai
  3. Pengaturan tampilan
  4. Format khusus (showData)
  5. Spesifikasi visual
Sesuaikan kategori dan periode waktu dengan kebutuhan analisis Anda. Pertimbangkan untuk membuat multiple charts untuk perbandingan antar periode.
3

Tips Umum Penggunaan Mermaid

Panduan umum untuk mengoptimalkan penggunaan diagram Mermaid.

Untuk semua jenis diagram Mermaid: 1. Persiapan - Tentukan tujuan diagram - Identifikasi audiens - Kumpulkan data/informasi - Pilih jenis diagram yang tepat 2. Struktur Prompt - Mulai dengan deskripsi jelas - Rinci komponen utama - Spesifikkan format - Tambahkan styling - Sertakan keterangan 3. Best Practices - Gunakan penamaan yang konsisten - Jaga kompleksitas tetap manageable - Manfaatkan grouping dan subgraphs - Tambahkan dokumentasi yang cukup - Test dan iterasi 4. Customization - Manfaatkan warna untuk emphasis - Gunakan styles yang sesuai konteks - Tambahkan notes untuk klarifikasi - Sesuaikan ukuran dan layout
Mulai dengan diagram sederhana dan tingkatkan kompleksitas secara bertahap sesuai kebutuhan. Selalu validasi dengan stakeholders untuk memastikan kejelasan dan kegunaan diagram.
