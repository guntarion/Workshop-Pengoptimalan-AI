1

Flowchart: Prosedur Pemeliharaan Preventif

Membuat diagram alir untuk prosedur pemeliharaan rutin peralatan pembangkit.

Tolong buatkan flowchart untuk prosedur pemeliharaan preventif menggunakan Mermaid dengan spesifikasi berikut: Komponen yang harus ada: - Jadwal pemeliharaan sebagai trigger - Persiapan dokumen dan peralatan - Pemeriksaan ketersediaan spare parts - Izin kerja dan safety briefing - Eksekusi pemeliharaan - Pengujian post-maintenance - Dokumentasi dan pelaporan Tambahan detail: - Percabangan untuk kondisi spare parts - Percabangan untuk hasil pengujian - Loop untuk perbaikan jika hasil test gagal - Subgraph untuk pengelompokan aktivitas terkait Gunakan format: flowchart TD dengan styling: - Kotak untuk proses normal - Diamond untuk decision points - Rounded edges untuk start/end - Warna berbeda untuk setiap subgraph - Arrow labels untuk detail kondisi

Struktur Prompt:

  1. Definisi jelas komponen utama
  2. Spesifikasi percabangan logika
  3. Pengelompokan aktivitas dalam subgraph
  4. Format dan arah diagram (TD = top-down)
  5. Styling spesifik untuk tiap jenis node
Sesuaikan dengan SOP pemeliharaan yang berlaku di fasilitas Anda. Tambahkan detail spesifik untuk equipment tertentu jika diperlukan.
2

Sequence Diagram: Protokol Penanganan Gangguan

Membuat diagram urutan untuk menggambarkan interaksi antar tim dalam penanganan gangguan.

Tolong buatkan sequence diagram untuk protokol penanganan gangguan pembangkit menggunakan Mermaid dengan spesifikasi berikut: Participants: - Operator Control Room - Supervisor Operasi - Tim Pemeliharaan - Tim Safety - Manajemen Plant - Sistem SCADA Alur interaksi: - Deteksi awal gangguan oleh SCADA - Notifikasi ke Operator - Eskalasi ke Supervisor - Mobilisasi Tim Pemeliharaan - Koordinasi dengan Tim Safety - Pelaporan ke Manajemen - Feedback setelah penanganan Detail tambahan: - Activation bars untuk menunjukkan periode aktif - Notes untuk informasi penting - Parallel actions saat diperlukan - Loop untuk monitoring berkelanjutan - Alt paths untuk skenario berbeda Gunakan format: sequenceDiagram dengan: - Arrows yang berbeda untuk tipe interaksi berbeda - Notes untuk detail prosedur - Participant boxes dengan warna berbeda

Struktur Prompt:

  1. Daftar lengkap participants
  2. Alur interaksi yang terurut
  3. Spesifikasi tipe interaksi
  4. Penggunaan activation bars
  5. Penambahan notes dan kondisi alternatif
Sesuaikan dengan struktur organisasi dan jalur komunikasi yang ada di perusahaan Anda.
3

Gantt Chart: Jadwal Overhaul Tahunan

Membuat diagram Gantt untuk perencanaan dan pelaksanaan overhaul pembangkit.

Tolong buatkan Gantt chart untuk jadwal overhaul tahunan pembangkit menggunakan Mermaid dengan spesifikasi berikut: Sections: - Persiapan - Pemadaman Unit - Pembongkaran - Inspeksi - Perbaikan - Pemasangan Kembali - Komisioning - Dokumentasi Tasks untuk setiap section: - Minimal 3 task per section - Durasi setiap task - Dependencies antar task - Milestone penting Tambahan: - Critical path highlighting - Weekend highlights - Progress indicators - Resource allocation Gunakan format: gantt dateFormat YYYY-MM-DD title Jadwal Overhaul Pembangkit excludes weekends Dengan: - Task relationships (after, before) - Section colors - Milestone markers - Progress indicators

Struktur Prompt:

  1. Pembagian sections yang jelas
  2. Detail tasks untuk setiap section
  3. Spesifikasi dependencies
  4. Format tanggal dan timeline
  5. Pengaturan display (colors, markers, etc.)
Sesuaikan timeline dengan jadwal pemeliharaan tahunan perusahaan dan pertimbangkan peak/off-peak seasons.
